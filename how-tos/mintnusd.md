# Mint NUSD

Minting NUSD using [USDe](https://www.coingecko.com/en/coins/ethena-usde) is now available at [bamk.fi/mint](https://bamk.fi/mint)

Minimum size is 10,000 NUSD and settlement can take up to 24 hours.&#x20;
